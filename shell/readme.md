# Shell module

This module holds configuration for a shell setup using `alacritty` as terminal emulator, `fish` as shell and `starship` as prompt.

## Requirements

The following binaries should be available:

 * fish
 * fisher
 * starship
 * fzf
 * zoxide
 * eza
 * ripgrep
 * bat
 * mosh
 * tmux
 * kubectl
